# Life Inbox Zero

The Smart Life Dashboard - A unified AI-powered life management system.

## Overview
Life Inbox Zero absorbs chaos, extracts actionable items, and auto-organizes them into a dashboard. It allows users to take actions with one click.

## Features
- **Input**: Screenshots, PDFs, Text, Links, Images, Emails.
- **Output**: Events, Tasks, Bills, Documents, Links, Reminders.
- **Agentic Automation**: Plans, decides, organizes, and executes MCP tools.
- **UI**: Card Garden UI with drag-and-drop zone and animated cards.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `adk run` (or `python main.py` if running manually)

## Structure
- `/app`: Core application code.
- `/google_adk`: ADK client wrappers.
- `/servers`: MCP servers for OCR and PDF extraction.
